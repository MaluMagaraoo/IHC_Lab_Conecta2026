# 🟣 Desafio IHC Lab: Vibe Coding para Protótipos Funcionais

O **desafio de Vibe Coding** em IHC (Interação Homem-Computador) convida estudantes e pesquisadores de UX a criar **protótipos funcionais de alta fidelidade em tempo recorde**, usando **inteligência artificial generativa**. O foco está na **intenção e experiência do usuário**, não na escrita manual de código. É a transição do “como codar” para o “o que criar”.

##  1. O Conceito

Vibe Coding em IHC consiste em **descrever fluxos de usuário ou wireframes em linguagem natural** para uma IA (ex.: Lovable, Bolt, v0.dev), que **transforma essas instruções em protótipos reais e navegáveis**.  
O objetivo é **experimentação criativa e satisfação imediata**, permitindo testes rápidos e ajustes instantâneos.


##  2. A “Vibe” na IHC (Foco no Usuário)

- **Design-First:** A IA gera o código enquanto o humano se concentra no **fluxo, espaçamento, cores e interação**.  
- **Iteração Rápida:** Feedbacks como *“torne este botão mais acessível”* ou *“mude o fluxo para a esquerda”* são aplicados em segundos via chat.  
- **Prototipagem de Alta Fidelidade:** Protótipos funcionais e realistas, ideais para testes de usabilidade, **superiores ao Figma estático**.


##  3. O Desafio Técnico

- **Prompt Engineering para IHC:** Traduzir intenções de design (heurísticas de Nielsen, acessibilidade) em instruções claras para a IA.  
- **Gestão da Vibe:** Saber quando intervir e pedir ajustes específicos para manter a **consistência do design**.  
- **Navegação e Estado:** Garantir que o protótipo gerado mantenha **estado entre telas** e funcione como uma aplicação real.


##  4. Vantagens no Contexto de IHC

- **Velocidade:** Tirar uma ideia do papel e colocar um protótipo nas mãos do usuário final em minutos.  
- **Acessibilidade:** UX Designers podem criar protótipos funcionais sem depender de desenvolvedores front-end.  
- **Feedback Imediato:** Permite rodar testes de usabilidade com **apps funcionais** e iterar com base no feedback real.


> **Resumo:** O desafio Vibe Coding é sobre **transformar intenções em experiências reais**, permitindo que inovação, acessibilidade e criatividade caminhem juntas no desenvolvimento de interfaces centradas no usuário.
